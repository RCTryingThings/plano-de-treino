# Plano de Treino — Iniciantes

Infográfico interativo em HTML com plano de treino para iniciantes, desenvolvido para o programa de acompanhamento online.

## Sobre

Plano estruturado para mulher, 50 anos, com condropatia patelar grau IV.  
Inclui dois treinos complementares (A e B) que estimulam o corpo na totalidade, com adaptações específicas para proteção articular do joelho.

## Conteúdo

- **Treino A** — Superior + Core (Push / Pull)
- **Treino B** — Inferior Adaptado + Core (Glúteos · Isquiotibiais · Estabilidade)
- Estrutura semanal e temporização
- Parâmetros de carga e cadência
- Progressão em 4 fases (12+ semanas)
- Adaptações e contraindicações para condropatia patelar

## Como usar

Abre o ficheiro `plano-de-treino-iniciantes.html` diretamente no browser.  
Não requer instalação nem dependências externas.

## Contexto

Criado no âmbito de um programa de personal training online com 12+ anos de experiência.  
Destinado a uso profissional por personal trainers certificados.

> ⚠️ Este plano não substitui acompanhamento médico ou fisioterapêutico.
